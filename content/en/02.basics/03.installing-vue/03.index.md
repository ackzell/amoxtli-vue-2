# Importing the lib

To get a hold of Vue.js, we can always do it like the in the old times.

You can see that we have an `index.html` and in there we have a `<script>` tag where we import the entire Vue.js lib. This is essentially a file that I downloaded and placed it where my `index.html` could read it too.

Feel free to check out the source code for Vue itself!

We'll explore the code itself in more detail later, don't worry. For now I just want you to know a few quick details on how to get the instance going.

```html file:/index.html showLineNumbers title="index.html" /vue-3.5.22.js/ {"Destructuring the lib to get the pieces we'll use in our code":9-10}
<script src="vue-3.5.22.js"></script>

<div id="app">
  <p>{{ message }}</p>
  <code>version: {{ version }}</code>
</div>

<script>
  const { createApp, ref, version } = Vue

  createApp({
    setup() {
      const message = ref('Hola Vue!')
      return {
        version,
        message,
      }
    },
  }).mount('#app')
</script>
```
