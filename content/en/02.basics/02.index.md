# Introduction

Let's start by installing the library in a couple different ways. This will be relevant for the sections that come after, but I wouldn't sweat it if you don't know these methods by heart.

The important part is you know _there is a way_ to do this manually, but the actual recommended way to start a new app is by using [create-vue](https://vuejs.org/guide/quick-start#creating-a-vue-application), the official tooling for node that will scaffold a Vue project for you with a lot of help.

Though if you want to stay in the browser, besides the [recommended ways to do so in the official docs](https://vuejs.org/guide/quick-start#try-vue-online), you can keep on exploring Amoxtli Vue (this platform), where I developed some magic that will enable you to try the basics of Vue right here on your browser effortlessly 🤓.

::warn
Please note that with the manual setup shown here we are not going to have :tooltip-trigger{id="sfc"}[SFCs available]. We need a _build step_ to make use of those. But we'll get there after this chapter.
::
