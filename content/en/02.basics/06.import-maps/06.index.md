# Import maps

The last thing we need to agree on: teaching the browser about the available modules.

This is achieved by using another `script` tag with a [new type](https://caniuse.com/import-maps): `importmap`. lajslk

```html file:/index.html showLineNumbers title="index.html" /type="importmap"/ {"aaa":1} {"New script tag":5-11}

```
