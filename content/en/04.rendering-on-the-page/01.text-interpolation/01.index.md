# Text Interpolation

> Text inter... what?

Just a fancy way to describe that you can use a Mustache syntax (`{{ }}`) to display dynamic content in your templates.

For example, that `message` variable we have been using in all the previous lessons, remember?

```vue showLineNumbers=false /{{/ /}}/
<h1>
{{ message }}
</h1>
```

But there is more you can do within those curly braces. You can run JavaScript expressions too, including method calls, arithmetic operations, and even :tooltip-trigger{id='globals'}[some globals] like `Math` or `Date` are available.

```vue {"We're familiar":2} {"Arithmetic expressions":8} {"Some globals are available":11} {"Conditional rendering using ternaries":14-18} /{{.+}}/
<template>
  <h1>{{ message }}</h1>

  <h1>{{ message.toUpperCase() }}</h1>

  Version is: <code>{{ version }}</code>

  <p>The answer is {{ 40 + 2 }}</p>

  <p>Random number: {{ Math.random() }}</p>
  <p>Current date: {{ new Date().toLocaleDateString() }}</p>

  <p>
    {{
      version.startsWith('3.')
        ? 'Using the latest Vue!'
        : 'This version is no longer officially supported'
    }}
  </p>
</template>
```

::warn
Keep in mind that [expressions **are not** statements](https://www.joshwcomeau.com/javascript/statements-vs-expressions/).
ie. _Inside a mustache_ you can't create variables (`const message = 'Hello!'`), or use control flow statements like `if`, `for`, etc.

```vue twoslash
// @errors: 2339 1005 1128

<script setup>

const message = ref('Interpolation')
const ok = true

console.log('version:', version)
</script>

<template>
  {{ if (ok) { return message } }}
</template>
```
