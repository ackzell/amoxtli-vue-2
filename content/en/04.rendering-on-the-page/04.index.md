# Intro

During this chapter, let's focus on the `<template>` tag. What things can be done and how can we control what is rendered on the page.

We'll explore some ways to render things conditionally, loop over data to render lists, and how to _bind_ attributes and properties to HTML elements.

There will be a little bit of code we need to write in the `<script>` tag, but we'll go deeper into that in the next chapter.
