# Intro

We've been printing stuff to the screen with interpolation and some built-in directives that are very useful.

We even styled our output a little bit already, but most of your apps will have dynamic content that the user is able to interact with. Otherwise there is no point in trying to use a UI framework such as Vue.

So let's now go deeper into what we can do inside of the `<script>` tag and how we can help our users interact with the apps we write.

We will probably be revisiting concepts we covered in [Chapter 3 - Rendering on the page](/basics/rendering-on-the-page/directives-intro) and add more stuff on top, so be sure to at least give a quick glance there if you aren't familiar with that.
