# Installing Vue

This is a recap. As the :tooltip-trigger{id="production-use"}[official docs point out], the recommendation is to use Vue with a build system, but you can also use it without one. In the next section, we'll see how to set up a project with a build system, but for now, let's see how to use Vue without one.

```vue title="App.vue" {"Handles the update from the input":6-8} {"This logs the 'wrapped' variable, eg. an object":11}  {"Whereas this logs the actual value we care about":12} {"Bind the message variable to the input's value and change it when user types in it":20}  collapse={2-5, 18-19, 29-30, 33-36}
<script setup>
import { ref, version } from 'vue'

const message = ref('Hola ref!')

function updateMessage(ev) {
  message.value = ev.target.value // `ev.target.value` is how you get the input's value natively, this is not Vue, it's the browser's API
}

function logMessage() {
  console.log(message)
  console.log(message.value)
}

console.log('version:', version)
</script>

<template>
  <h1>{{ message }}</h1>
  <input :value="message" :onInput="updateMessage">
  <button :onClick="logMessage">
    log message
  </button>

  <p>Version is: <code>{{ version }}</code></p>
</template>

<style>
h1 {
  color: #42b983;
}

button {
  margin-left: 0.5rem;
}
</style>
```
